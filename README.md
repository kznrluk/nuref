# Power Refs
Webベースの資料ビューアです。PCにある資料をウィンドウ内で自由に動かしたりリサイズしたりすることができます。

![Preview](https://github.com/kznrluk/power-refs/blob/main/docs/img.png?raw=true)

インターネット上に画像をアップロードせず、読み込み後はオフラインで動作します。
