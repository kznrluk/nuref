ブラウザだけで使えるクリエイター向け参考資料ビューア。

- 👩‍💻ブラウザでアクセスするだけですぐに使える
- 🐁ドラッグアンドドロップで画像を簡単追加
- 🗂️大量の画像も整理しやすく
- 🎶簡単・直感的な操作
- 💕アップロードなし アカウントなし 安全安心
